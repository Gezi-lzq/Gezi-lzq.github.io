(CS106)安装Qt Creator

Posted on 2022-02-15 Edited on 2022-02-28 In CS106 Views:

步骤

下载Qt installer

https://www.qt.io/download-qt-installer

运行安装程序

第一步需要注册一个账号，然后下面大多都是Next or Agree
其中需要注意的是


选择Qt 6.x for desktop development

如果没有这个选项，参考官方教程
If you don’t see Qt 6.x for desktop development as an option,click Custom installation and then proceed to the next window via “Next.” When you’re asked to “Select components” please only click MinGW 9.0.0 64-bit under Qt -> Qt6.2.2. Once again, you’ll click Qt and then under Qt 6.2.2 you’ll check the box for MinGW 9.0.0 64-bit.

安装CS106专用包

安装 Qt 后，您必须安装特定于 CS106 的包并执行完整的构建和运行循环以确认一切正常。

下载CS106包并解压

  • 下载此存档文件：📦 CS106.zip
  • 将下载内容解压缩（在 Windows 计算机上，单击“全部l解压缩”）到您选择的位置。您应该有一个文件夹CS106，其中包含多个文件和文件夹。

注意：路径内不要存在中文

打开并配置 CS106 项目

Qt Creator 项目包含一个以扩展名命名的文件。双击该文件会在 Qt Creator 中打开该项目。首次打开项目时，Qt Creator 会要求您配置项目构建工具包。**.pro **

  • 找到文件并立即打开它。 CS106.pro
    • 如果您的 Windows 文件资源管理器选项设置为隐藏文件扩展名，则文件将显示名称. 您可以通过选择菜单项“文件”->“选项”来更改是否在文件资源管理器中显示扩展名，选择“查看”选项卡，然后在“高级设置”下，取消选中“隐藏已知文件类型的扩展名”。单击“应用”按钮。CS106.pro
    • “配置项目”面板将显示可用构建套件的列表（请参见下面的屏幕截图）。应该已经选择了默认套件；它将与您在安装 Qt Creator（版本Qt 6.xx）时选择的桌面套件相匹配。
  • 通过单击“配置项目”按钮接受默认值。


如果您的 Qt Creator 显示没有可用的工具包，请查看Qt 安装说明。如果您错过了选择正确的选项，您可以重复这些步骤来重新安装 Qt。
